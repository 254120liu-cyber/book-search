# 搜书神器 — 使用说明书

---

## 一、你（运营者）每天怎么启动

### 准备工作（只需做一次）

1. 打开 Sakuracat 客户端，确认代理已开启（端口 7897）
2. Sakuracat 设为开机自启

### 每天启动（2 步）

**第 1 步**：打开 PowerShell，复制粘贴运行：

```powershell
C:\Users\25437\AppData\Local\Programs\Python\Python311\python.exe C:\Users\25437\Desktop\电子书\server.py
```

**第 2 步**：看到 `公网地址: https://xxx.trycloudflare.com` 后，把这个地址发给同学。

> ⚠️ **不要关这个 PowerShell 窗口！** 关了网站就挂了。

### 每天晚上

直接关 PowerShell 窗口就行。

## 二、用户（同学）怎么用

1. 手机浏览器打开你发的网址
2. 输入书名、作者或关键词 → 点搜索
3. 看到搜索结果，点击「获取下载」→ 跳转到 Z-Library 详情页

### 搜索技巧

| 搜什么 | 怎么搜 |
|--------|--------|
| 已知书名 | 输入完整书名，如「平凡的世界」 |
| 模糊搜索 | 输入部分关键词，如「平凡」 |
| 中英混合 | 直接输入，如「Python数据挖掘」 |
| 作者名 | 输入作者，如「余华」「刘慈欣」 |
| 教材 | 输入完整名称，如「高等数学 同济」 |

## 三、重要说明

### 地址会变吗？

每次重启 `server.py`，地址会变。需要把新地址发给同学。

想固定地址 → 买一个域名（约 1 元/年），配 Cloudflare Tunnel。

### 速度

每次搜索约 1-3 秒，取决于你的网络。

### 需要一直开着电脑吗？

是的。你白天开机，同学就能搜。晚上关机，网站不可用（同学也在睡觉，无所谓）。

### 能扛多少人？

Flask 单线程模式，大约能扛 **20-50 个同学**同时搜。再多会排队。

## 四、故障排查

| 现象 | 原因 | 解决 |
|------|------|------|
| 搜什么都 0 结果 | Sakuracat 没开 | 打开 Sakuracat |
| 网站打不开 | server.py 被关了 | 重新运行启动命令 |
| 搜索很慢 | 网络波动 | 等几秒重试 |
| `cloudflared.exe 未找到` | 文件位置不对 | 把 cloudflared.exe 放到电子书目录下 |
